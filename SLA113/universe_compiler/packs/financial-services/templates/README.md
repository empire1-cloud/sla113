# Templates

Reference materials, slide templates, and report templates for financial services.

## Sources

Mapped from Anthropic financial-services reference architecture:

- **pitch-deck/reference/** → Slide templates, formatting standards, calculation standards, XML reference for python-pptx
- **initiating-coverage/assets/** → Report templates, quality checklists
- **earnings-analysis/references/** → Best practices, report structures, workflows

## White-Label Awareness

Templates are the primary white-label surface — each ArchiSynapse partner
can have its own branded slide masters, report formats, and calculation
standards. Template selection should be driven by tenant configuration
at the product layer, not embedded in skills.
