# First-run checklist

1. Extract `ASSETS/shield_badge.png` from the approved Empire-1 source logo.
2. Run both validation-only commands.
3. Generate only `ocelotl_jaguar_shark` first.
4. Pack and install that sheet.
5. Inspect transparency, silhouette, badge scale, animation continuity, and collision alignment in-game.
6. Approve or revise the style contract before generating the remaining roster.
7. Generate the full roster with `--resume`.
8. Pack `--all --aggregate` and install `aztec_fish_species_v2.png`.
9. Generate static cabinet art in small reviewed batches.
10. Wire approved background, cannon, HUD, and VFX assets into the engine and run desktop/mobile smoke tests.
