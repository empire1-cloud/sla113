SLA113 :: Creature Animation System

character-agnostic atlas + state machine demo — placeholder art, real Phaser 3 engine

Idle / Swim / Attack / Hit / Enraged / Death states are driven entirely by the atlas contract in docs/ATLAS_CONTRACT.md — no per-creature animation code exists in src/. Fire with click/tap. Bosses rotate on kill and each wave scales minion/boss health & speed. A jade treasure koi occasionally swims through and flees the cannon — kill it before it escapes for a big payout. Chain kills within 1.5s to build a combo; x3+ upgrades your shots to harpoons. Coins persist across reloads. Placeholder sprites are procedurally generated by tools/pack_atlas.py — swap assets/atlases/<creature_id>.png/json with real art, no code changes required.
